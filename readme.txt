HOW TO USE :
- HOME PAGE IS AVAILABLE FOR EVERYONE, NO NEED TO BE LOGGED IN
- USERS SHOULD FIRST REGISTER ACCOUNT AND LOGIN TO USE THE OTHER FEATURES

AS BUYER :
- USERS ARE ABLE TO BUY GOODS FROM OTHER USER'S STORE
- PRODUCTS CAN BE ADDED INTO CART BEFORE CHECK OUT
- ONCE CHECK OUT, IT WILL BE SHOWN IN THE "BUYER" ORDER LISTS

AS SELLER :
- USERS ARE ABLE TO SETUP THEIR STORES ON THE SELLER PAGE AFTER LOGGED IN
- STORES NAME CAN NOT BE CHANGED AFTER IT IS SUCCESFULLY CREATED
- SELLERS CAN UPLOAD THEIR PRODUCTS INTO JUALANYUK PLATFORM AND WILL BE AUTOMATICALLY SHOWN ON EVERYBODY'S HOMEPAGE
- WHEN THERE IS AN ORDER FROM THE BUYER, IT WILL AUTOMATICALLY APPEARS ON THE "SELLER" ORDER LISTS
